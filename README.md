# 🎹 Piano JS - Vanilla Web Audio

Una aplicación web de teclado musical interactiva construida exclusivamente con **HTML5, CSS3 y JavaScript Vanilla**. Utiliza la **Web Audio API** para generar sonidos sintéticos mediante osciladores, sin necesidad de librerías externas ni archivos de audio pesados.

¡Listo para ser alojado en **GitHub Pages**!

## 🚀 Demo
Puedes ver la aplicación funcionando aquí: 
`https://jvilla86.github.io/piano/`

## ✨ Características
- **Sonido Sintético:** Generación de ondas de sonido en tiempo real (Osciladores).
- **Control por Teclado:** Mapeo de teclas físicas (A, S, D, F...) para tocar como en un DAW.
- **Diseño Responsivo:** Teclado adaptable a diferentes tamaños de pantalla.
- **Sin Dependencias:** Código limpio en JS puro (Vanilla).
- **Mapeo de Frecuencias:** Notas afinadas mediante frecuencias exactas en Hz.

## 🎹 Mapeo del Teclado
| Nota Musical | Tecla PC |
| :--- | :--- |
| Do (C) | `A` |
| Re (D) | `S` |
| Mi (E) | `D` |
| Fa (F) | `F` |
| Sol (G) | `G` |
| La (A) | `H` |
| Si (B) | `J` |
| *Sostenidos* | `W, E, T, Y, U` |
